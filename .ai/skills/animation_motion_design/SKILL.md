# 🛠️ animation_motion_design

**Purpose**: Professional micro-interactions, keyframe strategy, and motion token unification.

## 📋 Protocols
1. **Motion Tokens**: All durations and easings must be tokenized (`--duration-fast`, `--ease-out`).
2. **Reduced Motion**: Respect `prefers-reduced-motion` media queries.
3. **Pulsing States**: Use for typing indicators and loading skeletons.

## 🛠️ Usage
- Defined in `tokens.css` under Tailwind v4 `@theme`.
