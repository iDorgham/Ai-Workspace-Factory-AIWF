## 3. Key Sequencing Decisions

- **GitHub README first (Day 1):** Developer credibility gates all other trust. If the README isn't updated, every other channel loses authority.
- **Launch announcement same day:** LinkedIn + X fire simultaneously with README so there's something to link to immediately.
- **MENA content at Day 7:** Allows 6 days to observe initial dev community reaction; Arabic pieces are then positioned as amplification, not introduction.
- **Deep-dive at Day 10:** HN / Dev.to only after the community has had a week to discover and ask questions organically.
