# `.ai/context`

This folder holds **workspace-local context** when present. For the **AIWF factory repository** itself, canonical orientation lives in the docs tree:

- **[docs/CONTEXT.md](../../docs/CONTEXT.md)** — tiers, design catalog, sync commands, compliance snapshot  
- **[docs/PRD.md](../../docs/PRD.md)** — full product requirements  

Downstream workspaces created via `/mat` or `factory_materialize.sh` may add `project-type.md` or other context files here per profile.
