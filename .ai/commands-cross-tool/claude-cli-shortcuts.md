# Claude CLI Shortcuts

Use this when running workspace commands through Claude CLI / Claude Code.

## Shortcut format

Paste this into Claude chat:

```text
Run Sovereign workspace command: <router_command>
Use `.ai/commands.md` as routing contract and follow its output format exactly.
```

Example:

```text
Run Sovereign workspace command: /create blog posts about sustainable interiors
Use `.ai/commands.md` as routing contract and follow its output format exactly.
```

## Canonical IDs

Use IDs from `.ai/commands-cross-tool/canonical-command-catalog.json` to pick command text.
