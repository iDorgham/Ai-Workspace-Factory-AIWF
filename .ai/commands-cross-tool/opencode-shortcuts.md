# OpenCode Shortcuts

Use this when running workspace commands through OpenCode CLI.

## Shortcut format

```bash
opencode --prompt "Run Sovereign workspace command: <router_command>. Follow .ai/commands.md output format." --model default --output json
```

Example:

```bash
opencode --prompt "Run Sovereign workspace command: /create project pages. Follow .ai/commands.md output format." --model default --output json
```
