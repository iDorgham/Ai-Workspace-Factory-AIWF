# Qwen Shortcuts

Use this when running workspace commands through Qwen CLI.

## Shortcut format

```bash
qwen --prompt "Run Sovereign workspace command: <router_command>. Follow .ai/commands.md output format." --max-tokens 8000 --output json
```

Example:

```bash
qwen --prompt "Run Sovereign workspace command: /sync. Follow .ai/commands.md output format." --max-tokens 8000 --output json
```
