# Logs & reports (documentation)

This folder groups **human-readable** phase reports, implementation guides, and archived run snapshots for the Sovereign workspace.

| Path | Purpose |
|------|---------|
| Top-level `*.md` here | Phase status, integration plans, day guides |
| `legacy-snapshots/` | Stub README only; historical files → **`archive/legacy-snapshots/`** at repo root |
| `milestones/` | Brief “ready for execution” notes and HTML milestone snapshots |

**Operational logs** (workflow, audits, path integrity) → `.ai/logs/`.
