# 06 — Brand Reference

**Purpose:** Sovereign studio identity, market positioning, and content voice guidelines

This folder is your single source of truth for brand consistency across all content.

---

## 📄 Core Documents

| File | Purpose | Audience |
|------|---------|----------|
| `market-positioning.md` | Studio niche, target clients, USPs | Strategy, positioning |
| `brand-voice/style-rules.md` | Writing tone, voice rules, grammar | Content creators |
| `brand-voice/glossary.md` | Approved terms, prohibited language | All writers |
| `brand-voice/tone-examples.md` | Voice examples by context | Learners, new writers |
| `brand-voice/voice-refinement.md` | Voice analysis, evolution tracking | Brand strategists |
| `brand-discovery.md` | Discovery framework and process | Onboarding |

---

## 🎨 Brand Voice Layers

### 1. Market Positioning (`market-positioning.md`)
- Studio location, niche, target clients
- Unique selling propositions (USPs)
- Market differentiation
- Service offerings

### 2. Style Rules (`brand-voice/style-rules.md`)
- Tone guidelines (formal/casual/friendly)
- Voice characteristics (authoritative/relatable/etc.)
- Grammar and punctuation rules
- Sentence structure preferences

### 3. Terminology (`brand-voice/glossary.md`)
- Approved vocabulary
- Prohibited terms
- Brand-specific language
- Industry terminology

### 4. Examples (`brand-voice/tone-examples.md`)
- Real samples by context
- Before/after comparisons
- Contextual voice shifts

### 5. Evolution (`brand-voice/voice-refinement.md`)
- Tracking brand voice development
- Analysis of effectiveness
- Refinement notes

---

## ✅ Content Creation Checklist

Before writing any content, verify:

1. **Market Positioning** — Am I writing for Sovereign's target clients?
2. **Style Rules** — Does my tone match the brand voice?
3. **Glossary** — Am I using approved terminology?
4. **Examples** — Do my sentences match the tone examples?

## 🎯 Common Tasks

### Creating New Content
1. Read `market-positioning.md` for audience context
2. Review `style-rules.md` for tone
3. Check `glossary.md` for approved terms
4. Reference `tone-examples.md` for samples
5. Write content
6. Verify against all 4 layers

### Updating Brand Voice
1. Document change in `voice-refinement.md`
2. Update `style-rules.md` if rules changed
3. Update `glossary.md` if terminology changed
4. Add new examples to `tone-examples.md`
5. Notify all content creators

### Onboarding New Writer
1. Have them read `market-positioning.md`
2. Study `brand-voice/style-rules.md`
3. Review `tone-examples.md` samples
4. Check `glossary.md` before writing
5. Practice with small pieces first

## 🎬 Example Workflow

**Writing a blog post about kitchen design:**

1. Check `market-positioning.md` → "Our target clients are luxury homeowners aged 35–55"
2. Review `style-rules.md` → "Tone: sophisticated but accessible, professional but warm"
3. Look at `glossary.md` → Use "bespoke design," not "custom," avoid "trendy"
4. Study `tone-examples.md` → See 2–3 actual Sovereign sentences
5. Write draft maintaining this voice
6. Self-review against glossary + style rules
7. Ready to publish ✅
