# Qualitative Coding & Analysis

## Methodology

### 1. Open Coding
- Identifying discrete concepts and categories in raw transcripts or feedback.
- Tags: `[PAIN] [GAIN] [UI-FRICTION] [VALUE-PROP]`.

### 2. Axial Coding
- Connecting categories and identifying relationships between them.
- Example: "Users who experience [UI-FRICTION] on mobile are 50% more likely to mention [COST] as a barrier."

### 3. Selective Coding
- Selecting the core category and systematically relating it to other categories to build a narrative.

## Implementation Pattern
- **Tag Cloud Generation**: Visualizing frequency of sentiment tags.
- **Thematic Extractor**: Pulling direct quotes that represent a larger trend.

## Anti-Patterns
- Confirmation bias (only coding data that supports the current vision).
- Over-quantifying small sample sizes.
