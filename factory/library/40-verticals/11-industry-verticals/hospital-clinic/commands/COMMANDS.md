# Hospital Clinic - Commands

> [!IMPORTANT]
> This vault follows the **Shortened Standard** for Hospital-Clinic operations.

| Command | Action |
|:---|:---|
| **/medical PatientIndex** | Scrape clinical files and generate anonymized patient stubs. |
| **/medical AuditLicenses** | Sentinel check for HCP license validity (DHA/DOH/MoHP). |
| **/medical AuditInventory**| Audit cold-chain pharmacy stock and narcotic registers. |
| **/medical EMR-Create** | Assemble a bilingual (AR/EN) clinical patient report. |
| **/finance ClaimAudit** | Scrape clinical files for missing insurance pre-authorizations. |
| **/medical CheckInteractions**| Audit prescription stubs for potential drug-drug interactions. |
