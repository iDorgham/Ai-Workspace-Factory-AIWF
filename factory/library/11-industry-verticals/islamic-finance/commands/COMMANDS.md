# Islamic Finance - Commands

> [!IMPORTANT]
> This vault follows the **Shortened Standard** for Islamic-Finance operations.

| Command | Action |
|:---|:---|
| **/sharia AuditContract** | Scan a financial instrument for Riba/Gharar (Interest/Uncertainty). |
| **/sharia Sukuk-Draft** | Scaffold an asset-backed Sukuk investment certificate. |
| **/sharia Zakat-Calc** | Execute corporate Zakat calculations and charity distribution. |
| **/finance Murabaha-Audit**| Verify cost-plus-profit transparency in asset financing. |
| **/sharia Screening** | Ethical screening of investment nodes against Sharia blacklists. |
| **/sharia Fatwa-Search** | Retrieve relevant Sharia Board decrees for a specific instrument. |
