# Lodging Hotel - Commands

> [!IMPORTANT]
> This vault follows the **Shortened Standard** for Lodging-Hotel operations.

| Command | Action |
|:---|:---|
| **/hotel BookingIndex** | Scrape and hash guest reservation stubs. |
| **/hotel YieldReport** | Generate a RevPAR/Occupancy performance summary. |
| **/hotel Concierge-Draft** | Assemble a bilingual (AR/EN) luxury guest itinerary. |
| **/hotel Facility-Audit** | Audit housekeeping and facility maintenance logs. |
| **/search Guests** | Standard operational trigger with guest-node filtering. |
| **/verify Room-Ready** | Automated check for room inspection hashes. |
