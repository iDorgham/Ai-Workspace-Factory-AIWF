# 🧬 Predictive Unit Economics

> **Governance:** 02-Commerce | **Sector:** 📊 Analysis | **Type:** Anchor Skill (Tier 3)

## 📋 Implementation Protocol

1. **CAC/LTV Ratio:** Build dynamic models to track regional customer acquisition efficiency.
2. **Cohort Analysis:** Segment regional users by spend, retention, and referral-loops.
3. **Forecasting Logic:** Apply Monte-Carlo simulations to predict market expansion outcomes.

## 🌍 Regional Calibration (MENA Context)

- **Cohort Behavior:** Account for high referral-rates (Waro-of-Mouth) in Arab social structures.
- **Economic Cycles:** Factor in regional liquidity shifts during religious seasons.

## 🛡️ Critical Failure Modes (Anti-Patterns)

- **Anti-Pattern:** Static Forecasting -> *Correction:* All models must handle multi-scenario volatility.
- **Anti-Pattern:** Ignoring regional payment-mix (COD/Tabby) -> *Correction:* Build cash-flow delays into the logic.
