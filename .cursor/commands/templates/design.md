# `/design` Catalog Usage

## Purpose

Browse and select imported `design.md` references from:
- `https://github.com/VoltAgent/awesome-design-md/tree/main/design-md`

## Commands

- `/design` or `/design list`  
  Show available design packs from `factory/library/design/README.md`.

- `/design use [pack]`  
  Select one pack for the current project context.

- `/design install [pack]`  
  Install/copy selected reference into template scope.

## Catalog Location

- Library source: `factory/library/design/[pack]/design.md`
- Template mirrors: `factory/library/templates/design/[pack]/design.md`
- Workspace mirror: `.ai/templates/design/[pack]/design.md`

## Suggested Chain

- `/design list`
- `/design use [pack]`
- `/create website pages`
