# 🏭 Autonomous Workspaces Directory

This directory houses all output environments generated by the AI Workspace Factory.

Unlike legacy "projects" which shared the same global AI resources, every subdirectory instantiated here operates as an **independent, 100% autonomous software environment**.

## Default Workspace Architecture
Every generated `<project-slug>/` contains:

- `.ai/` - Local agent registries, memory state, and workflow logs mapped exclusively to this project.
- `docs/` - Auto-generated documentation, PRDs, and registry manifests.
- `.cursor/`, `.windsurf/` (or equivalents) - Pre-configured rules and tasks ensuring it works identically across AI IDEs.
- `CLAUDE.md` / `AGENTS.md` - Self-contained capability configurations so standard CLIs and plugins function instantly.
- **Clean Root** - The actual application logic (e.g., `src/`, `config/`) lives cleanly in the root of the workspace without interference from the Factory generator.

To spin up a new integrated workspace, utilize the factory composer script.
