## 🆕 New Features

### `/deploy` Command (Explicit Vercel Deployment)
The most critical new addition. Vercel deployment is **ONLY** triggered when `/deploy` is explicitly invoked. It will never auto-trigger on Git push or merge.

**Flags**: `--prod`, `--preview`, `--env=KEY=VALUE`, `--dry-run`, `--silent`, `--region`
**Steps**: Vercel CLI check → project link → env pull → build → deploy → log to `.ai/logs/deployments.log`

### `/do` Command (Idea → PRD)
Structured transformation of a raw idea into a formal PRD with REQ-IDs, acceptance criteria, constraints, and regional notes. Replaces ad-hoc PRD generation.

### `/plan` Command (SDD Blueprint)
Generates a complete AIWF SDD workspace from a PRD with validated `spec.yaml` contracts, Mermaid architecture, database schema, task graphs, and regional compliance folders.

### New SDD Blueprint Folder Structure
Every `/plan` output follows the strict `plan/` blueprint with `_manifest.yaml`, `spec.md`, `spec.yaml`, `design.md`, `contracts/`, `regional/`, and `validation/` folders.

### `spec.yaml` Schema Enforcement
Machine-readable contract with `phase_id`, `acceptance_criteria` (with `test_fixture` and `contract_gate`), `regional_compliance`, `approved_by`, and `evolution_hash` fields. Enforced by Contract Guardian sub-agent.

## ⬆️ Upgraded Features

### Healing Bot → v2 (Predictive)
- v1: Reactive structural repair on detected drift.
- v2: Predictive monitoring — detects and flags drift patterns *before* failure, enabling preemptive remediation.

### Guide Protocol — Instructor Mode
- Now **mandatory on every response** (not optional).
- New format: `✅ Done` · `📚 Learn` · `▶️ Next` · `💡 Suggest`.
- Includes copy-button support for generated prompts and specs.

### Brainstorm — Structured & Copyable
- Outputs now include Copyable Prompt Blocks (fenced `yaml`/`markdown`).
- Probabilistic trigger (~15-20%) during active development when regional or compliance gaps are detected.

### Regional Engine — Fully Activated
- `--region=egypt|redsea|mena` flag available across ALL commands.
- Regional Adapter sub-agent handles RTL, Fawry/Vodafone Cash, Law 151/2020, EGP/SAR/AED.
- Regional compliance is a hard gate for auto-merge eligibility.

### Auto-Merge — Stricter Gates
All 5 gates must pass: tests ✓ + coverage ≥100% ✓ + Omega Gate approval ✓ + regional compliance ✓ + no pending mutations ✓.
